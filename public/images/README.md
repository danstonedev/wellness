# Patient & Case Study Images

Add your images here. They will be available at `/wellness/images/filename.jpg` in your app.

## Usage in Components

```tsx
// In any component
<img src="/wellness/images/eleanor.jpg" alt="Eleanor" />
```

## Folder Structure

Organize images by case study:
```
images/
├── geriatrics/
│   └── eleanor.jpg
├── neurology/
│   └── marcus.jpg
├── pediatrics/
│   └── leo.jpg
└── sci-homeless/
    └── sam.jpg
```

## Supported Formats
- `.jpg`, `.jpeg`
- `.png`
- `.webp` (recommended for web)
- `.svg`
